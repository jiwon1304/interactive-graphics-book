# 카툰/툰(NPR) 렌더링 토픽 카탈로그 (집필 백로그)

셀 셰이딩·아웃라인·NPR 커스텀 라이팅·matcap·텍스처 채널·스타일라이즈드 PBR·회화풍, 그리고 프로덕션 케이스.
대부분 **3D 셰이더 데모**(DemoCanvas + 커스텀 GLSL, 간단 지오메트리)로 구현 — "과정>결과" 위젯(램프 에디터,
아웃라인 비교, matcap UV 룩업, 채널 언패커 등). 병렬 리서치(2026-06)·중복 제거.
표기: `[난이도] 한글명 (English) — 데모 힌트`. (집필 시 [AUTHORING-GUIDE.md](./AUTHORING-GUIDE.md) 준수)

## A. 셀 셰이딩 & 램프 라이팅
1. `[입문]` 하드 스텝 디퓨즈 (Hard step `step(NdotL)`) — 임계값 슬라이더로 명암 경계가 표면 위 이동
2. `[입문]` 밴드 수 / 양자화 (Quantized banding) — `floor(NdotL·N)/N`, 밴드 수 2→8 + 명암 막대 계단화
3. `[중급]` 임계값 조절 비균등 밴딩 — 명암 막대 위 드래그 핸들로 그림자 면적 직접 정함
4. `[중급]` 1D 램프 / 그라디언트 LUT — 편집 가능한 가로 램프 바, 스탑 이동 시 구가 즉시 re-band (핵심)
5. `[중급]` 색조 시프트 램프 (Warm-cool ramp) — 그림자=차가운색/하이라이트=따뜻한색, "밝기만 vs 색조" 토글
6. `[입문]` 하프 램버트 (Half-Lambert, Valve) — `NdotL·0.5+0.5`, 람베르트와 분할 비교 + remap 곡선
7. `[중급]` 라이트 랩 (Light wrap / wrap diffuse) — wrap 슬라이더로 터미네이터가 표면 위 미끄러짐
8. `[심화]` 에너지 보존 하프 램버트 — remap 뒤 `pow` 보정, 보존 on/off 비교
9. `[중급]` 스무스스텝 밴드 경계 — soft↔hard 슬라이더로 경계 폭, 경계 영역 하이라이트
10. `[심화]` fwidth 경계 안티앨리어싱 — 화면 공간 일정 폭, 줌·비스듬한 면에서 계단 제거
11. `[중급]` 그림자 색조/틴트 (Colored shadow) — lit색·shadow색 픽커, 곱하기 vs 별도 컬러 lerp
12. `[심화]` 터미네이터 따뜻한 띠 (Terminator warm line) — 경계에 채도 높은 좁은 3번째 밴드
13. `[중급]` 토온 앰비언트 / 필 — 가장 어두운 밴드 들어올리기 = 램프 좌단 스탑 올리기(동치 시연)
14. `[심화]` AO/셀프섀도 마스크를 램프에 (AO-driven shadow) — 베이크 AO 영향도로 오목부에 그림자 추가(GG Xrd식)
15. `[심화]` 다중 광원 밴딩 — 방향광/점광에 다른 밴드 수, 점광 드래그 시 toon 스포트라이트

## B. 외곽선 / 라인아트 (Outlines)
16. `[입문]` 반전 외피(백페이스) 아웃라인 (Inverted-hull) — 부푼 복제 메시 + front-face cull 단계 토글
17. `[입문]` 노멀 압출 vs 위치 압출 — 압출 벡터(화살표) 노멀/방사형 전환, 코너 차이
18. `[중급]` 하드엣지 노멀 균열 문제 — split normal에서 윤곽 갈라짐, 정점 노멀 화살표 시각화
19. `[심화]` 스무딩/베이크 노멀 (Guilty Gear trick) — 보조 속성의 평활 노멀로 압출, 균열 제거 비교
20. `[중급]` 스크린 공간 일정 두께 — 거리 보정, 멀어져도 픽셀 두께 일정 (월드 두께와 비교)
21. `[심화]` FOV 보정 두께 — 줌 시 두께 출렁임 보정 on/off
22. `[중급]` 정점 컬러/마스크 두께 제어 — 부위별 두께 페인트(얼굴 얇게·실루엣 굵게), GG 핵심
23. `[심화]` 깊이 바이어스 & 라인 숨김 — 윤곽을 앞/뒤로 밀거나 영역별 끄기
24. `[중급]` Sobel 깊이/노멀 엣지 후처리 — 마우스 따라다니는 3×3 커널 윈도우로 가중치·합산·임계 표시
25. `[중급]` 깊이 vs 노멀 불연속 — "깊이만/노멀만/합침" 3-way로 각자 놓치는 엣지 확인
26. `[심화]` 비선형 깊이 임계값 보정 — 가까이/멀리 같은 임계가 다르게 작동, 깊이로 임계 변조
27. `[심화]` 서피스 ID 버퍼 엣지 — 같은 면에 겹친 두 오브젝트를 ID 불연속으로 분리(WebGL)
28. `[입문]` 거리 기반 윤곽 페이드 — 카메라 거리로 알파 start→end 선형 사라짐
29. `[중급]` 베이스 컬러 틴트 윤곽 — 검정 대신 표면색×어둡기, 부위별 라인색
30. `[심화]` 스케치/해칭 라인 — 노이즈·해칭·흔들림으로 손그림 펜선
31. `[최신]` 시사 컨투어 / 어패런트 릿지 — 시점 의존 곡률로 곡면 형태선(실루엣 너머)
32. `[입문]` 절차적 라인아트 (Blender Freestyle/Grease Pencil) — silhouette/crease/material 엣지 타입별 토글

## C. NPR 커스텀 라이팅 & 하이라이트
33. `[중급]` 커스텀 램프 BRDF / 라이팅 LUT — 편집 곡선이 곧 라이팅 응답, 계단=셀/부드러움=페인터리
34. `[입문]` 림 라이트 / 프레넬 엣지 (Rim/Fresnel) — `pow(1-NdotV,p)`, rim power·color + `1-NdotV` 마스크 오버레이
35. `[중급]` 구치 웜–쿨 셰이딩 (Gooch) — warm/cool 두 극 lerp, 람베르트↔구치 슬라이더
36. `[입문]` 하드 셀 스펙큘러 — Blinn에 smoothstep 임계, 둥근 하이라이트가 하드 도형으로 "굳음"
37. `[심화]` 아니메 하이라이트 셰이핑 (Anjyo) — translate/rotate/squash/split로 블롭 하이라이트 빌더
38. `[심화]` 머리카락 이방성 하이라이트 (Anisotropic) — 탄젠트 방향·시프트·스텝으로 천사고리 띠
39. `[중급]` 멀티밴드 셰도우 (Genshin 1차/2차) — 두 임계+밴드 컬러로 명암 다단 분할
40. `[중급]` 커스텀 그림자색 & 가짜 SSS 엣지 — 경계 양쪽 따뜻한 띠로 값싼 표면하산란 흉내
41. `[심화]` 노멀 오버라이드 / 페이스 노멀 — 구형/평활 노멀로 NdotL, 지저분한 얼굴 자기그림자 정리
42. `[심화]` 아트 디렉티드 가짜 라이트 방향 (GG Xrd) — 오브젝트별 라이트 벡터 주입, 씬 라이트와 분리
43. `[중급]` 하프톤/스크린톤 라이팅 — NdotL을 망점 격자와 비교, dot 크기·각도·임계
44. `[중급]` 크로스해치 라이팅 (TAM) — 명도 구간별 해칭 레이어 누적·교차
45. `[중급]` 스펙큘러 마스크 & 톤맵 컨트롤 — 마스크로 부위별 광택, 톤맵 곡선 에디터
46. `[심화]` BotW식 뷰 의존 스펙큘러 & 토온 AO — 시점 따라 스펙 띠 미끄러짐, AO를 컬러 그림자로 램프

## D. matcap & 텍스처 채널 아트 컨트롤
47. `[입문]` matcap 기초: 뷰공간 노멀→구 UV — 프로브 노멀 화살표가 구 이미지의 텍셀로 선 긋기(룩업 시각화)
48. `[입문]` matcap 굽기: 조명+재질을 한 장에 — 라이트·하이라이트 조작 시 matcap 이미지가 재-굽힘 + 메시 반영
49. `[중급]` matcap 합성: 가산 vs 곱셈 — base·mul·add 3트랙 누적, `base·mul + add`
50. `[중급]` 더블 matcap: 명부/암부 분리 — NdotL 마스크로 lit/shadow matcap 블렌드
51. `[심화]` matcap 회전 아티팩트 & 좌표공간 — 뷰공간(따라 돔) vs 월드 보정(고정) 비교
52. `[중급]` matcap으로 가짜 SSS·금속·이방성 — 림 워밍/수평 스미어가 효과 만드는 핫스팟 표시
53. `[중급]` 눈·피부 전용 matcap — 안구 캐치라이트 이동 + 패럴랙스 굴절
54. `[입문]` RGBA 채널 패킹 — 채널 탭 시 AO/그림자마스크/스펙/스무스 역할 분리(언패커)
55. `[심화]` 얼굴 SDF 그림자 맵 (Genshin/Honkai) — 라이트 방위각 다이얼로 SDF 임계선이 코·앞머리 그림자 스윕
56. `[중급]` 머티리얼 ID / 영역 마스크 — RGB 영역 탭 시 부위별 램프·아웃라인·스펙 격리
57. `[심화]` 머리카락 시프트·플로우 맵 — 시프트로 1·2차 하이라이트 이동, 플로우로 결 따라 휨
58. `[중급]` 버텍스 컬러 아트 컨트롤 — R=AO·G=그림자바이어스·B=아웃라인폭 페인트
59. `[심화]` 해칭/임계값 맵 (TAM) — 톤 내리면 해칭→크로스해치 단계 전환
60. `[입문]` 2D matcap 디버거 — matcap 텍셀↔3D 노멀 방향 양방향 잇는 순수 2D 다이어그램

## E. 스타일라이즈드 PBR & 회화/일러스트 NPR
61. `[중급]` 스타일라이즈드 PBR / PBNPR — PBR↔스타일 슬라이더, 에너지 보존 깨지는 순간 적색 경고
62. `[입문]` 그래디언트 매핑 (Luminance→ramp) — 휘도 기준 재색화, 휘도 0=좌·1=우 매핑 화살표
63. `[입문]` 포스터화 / 플랫 컬러 — steps 슬라이더로 그라데이션 띠 분리, RGB별 vs 휘도 양자화
64. `[심화]` 해칭 & 크로스해치 (TAM) — 어두울수록 선 촘촘, 45°/90° 레이어 겹침
65. `[중급]` 스티플링 (Stippling) — 블루노이즈 점이 톤 비례 임계로 켜짐, 밀도∝1/톤
66. `[중급]` 하프톤 / 벤데이 도트 — 셀 크기·각도·임계로 점 격자, 어두울수록 점 큼
67. `[심화]` 스파이더버스 룩 — halftone→hatch→색수차→linework 레이어 스택 누적
68. `[심화]` 수채화 렌더링 — 엣지 다크닝/granulation/종이/흔들림 4효과 분리 토글
69. `[심화]` 유화 브러시스트로크 (Painterly) — 큰→작은 스트로크 레이어, 엣지/광원 방향 정렬
70. `[심화]` 이방성 쿠와하라 필터 — 구조 텐서 흐름 따라 평탄화, 흐름장 화살표 + 분산 시각화
71. `[심화]` XDoG (eXtended DoG) — σ·비율·샤프닝·임계로 라인↔목판↔파스텔, 중간 버퍼 노출
72. `[최신]` 플로우 기반 추상화 / ETF — 엣지 탄젠트 플로우를 LIC 스트림라인으로, 평활 반복
73. `[중급]` 스타일용 컬러 그레이딩 (3D LUT) — 룩 전환 + RGB 큐브에서 색 이동 화살표
74. `[최신]` 셰이딩 리그 (Art-directable shading) — 하이라이트 편집 프리미티브, 라이트 회전 시 모양 보간
